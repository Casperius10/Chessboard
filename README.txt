This project is the firmware for a chessboard powered by an Arduino Uno. All needed code, libraries, and hardware requirements are included. The code for the board itself is built as a PlatformIO project (VSC extension required)

USAGE NOTES (put these somewhere else later):
- Castling: pick up king, then rook, then place both
- En passant: pick up and place down attacking pawn, then remove attacked pawn
